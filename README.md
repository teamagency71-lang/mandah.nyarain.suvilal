# Mandah Nyarain Suvilal

Static website for Mandah Nyarain Suvilal.

## Deploy on Netlify

Use these settings when importing the GitHub repository into Netlify:

- Branch: `main`
- Base directory: leave empty
- Build command: leave empty
- Publish directory: `.`

The site entry file is `index.html`, and all images are in `assets/`.

## Pages

- `index.html` - public website
- `admin.html` - admin dashboard
- `doctor.html` - doctor dashboard
- `cook.html` - cook dashboard
- `profile.html` - user profile
